# AI 文化讲解文本模板

本模板供后端 `generator.py` 使用，将 `patterns.json` 中对应字段填入占位符，生成稳定的文化讲解文本。

---

## 模板格式

```
系统识别该图片可能包含【name】元素。

从视觉特征看，该纹样通常表现为【visual_features】

在中国玉文化中，【name】常被理解为【cultural_meaning】

这类纹样常见于【common_objects】，与【periods】的审美和礼制观念有关。

鉴赏时可以重点观察【appreciation_tips】
```

---

## 填充示例：龙纹

```
系统识别该图片可能包含龙纹元素。

从视觉特征看，该纹样通常表现为身体盘曲、线条流动，常带有兽首、卷尾或云气装饰，整体呈S形或C形蜿蜒造型。

在中国玉文化中，龙纹常被理解为祥瑞、权力与礼制的象征，是中国玉器中最具代表性的传统纹样，代表帝王权威与天地沟通。

这类纹样常见于玉佩、玉璧、玉带钩、玉饰件，与春秋战国、汉代、明清的审美和礼制观念有关。

鉴赏时可以重点观察龙首造型、身形曲线、线条节奏，以及纹样与器物功能之间的关系。
```

## 填充示例：谷纹

```
系统识别该图片可能包含谷纹元素。

从视觉特征看，该纹样通常表现为圆形凸起颗粒均匀密布于器物表面，形似发芽的谷粒，颗粒饱满、排列规整，有时顶端带有螺旋小尾。

在中国玉文化中，谷纹常被理解为五谷丰登、农业丰收的象征，寄托对粮食充足、生活富足的美好愿望，是战汉玉璧上最常见的吉祥纹样。

这类纹样常见于玉璧、玉环、玉玦，与战国、汉代的审美和礼制观念有关。

鉴赏时可以重点观察谷粒凸起的饱满程度和排列的均匀性，感受其朴素自然的农耕文化内涵。
```

---

## 使用说明

- 占位符全部从 `patterns.json` 对应条目中读取，无需人工填写
- `common_objects` 为数组，填充时以顿号连接，如：`玉佩、玉璧、玉带钩`
- `periods` 为数组，填充时以顿号连接，如：`春秋战国、汉代、明清`
- 结果表述使用"可能包含"而非"确定是"，避免伪装成权威鉴定
- 模板固定，不依赖大模型实时生成，确保演示稳定性
