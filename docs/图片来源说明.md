# 图片来源说明

本文档记录项目中所有示例图片及文字资料的来源，供提交审查使用。

---

## 使用原则

- 优先使用公开博物馆数字资源、学术课程资料及知识共享（CC）授权图片
- 所有图片仅用于课程学习展示，不作商业用途
- 正式使用前需将占位图替换为已确认来源的真实图片
