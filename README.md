# 玉纹识 —— AI 玉器纹样识别与文化解读系统

## 项目简介

本项目是一个展示版 AI 辅助玉器鉴赏原型系统，支持上传玉器图片、识别传统纹样、展示文化解读与相似案例推荐。

## 团队分工

| 成员 | 角色 |
|------|------|
| 成员 A | 前端界面与演示体验 |
| 成员 B | 简化后端与系统联调 |
| 成员 C | 玉文化数据、PPT 与视频整合 |

## 目录结构

```
yuwen-shi/

│   ├── patterns.json
```
